### 数组扁平化

#### 原理

多维数组拍平成一维数组

#### 实现

```js
Array.prototype.myFlat = function () {
  return this.reduce((pre, item) => {
    return pre.concat(Array.isArray(item) ? item.myFlat() : item);
  }, []);
};
```

控制层数

```js
Array.prototype.myFlat = function (dep) {
  return this.reduce((pre, item) => {
    return pre.concat(
      Array.isArray(item) && dep > 0 ? item.myFlat(dep - 1) : item
    );
  }, []);
};
```
