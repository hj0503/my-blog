9. 二叉树

#### 常见的时间复杂度

常量阶 O(1)、指数阶 O(2 的 n 次方)、对数阶 O(logn)、阶乘阶 O(n!)、线性阶 O(n)、线性对数阶 O(nlogn)、平方阶 O(n*\n)、立方阶 O(n*n\*n)、K 次方阶 O(n 的 k 次方)

#### 常见数据结构

数组、链表、栈、队列、散列表、二叉树、堆、跳表、图、Trie 树

#### 常见算法

递归、排序、二分查找、搜索、哈希算法、贪心算法、分治算法、回溯算法、动态规划、字符串匹配
